# Assetless Source Export Plugin

Godot addon that exports source code with blank game assets

![GitHub Logo](./preview.png)

## Features
- [x] Images
	- [x] bmp
	- [x] dds
	- [x] exr
	- [x] jpg
	- [x] ktx
	- [x] png
	- [x] tga
	- [x] webp
- [x] SVG
- [x] Resources 
- [x] Settings cache
- [ ] Swap files
- [ ] Random colors
- [ ] Audio
